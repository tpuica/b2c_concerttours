b2c_concerttours 

 Platform Setup:
   1. Setup platform using command ./install.sh -r b2c_concerttours setup -A initAdminPassword=[password]
   2. Initialize platform using command ./install.sh -r b2c_concerttours initialize -A initAdminPassword=[password]
   3. Start platform using command ./install.sh -r b2c_concerttours start
